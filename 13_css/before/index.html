Приклад 1 - first-line

Загальні відомості:
Задає стиль першої стрічки тексту.
Можна змінювати лише властивості шрифту та тексту (колір, розмір)
css: .text_fln { line-height: 32px; font-size: 32px; margin: 0 0 30px 0; }
.text_fln::first-line{ color: green; }
.list_fln li { font-size: 24px; margin: 0px 0px 20px 0px; }

Приклад 3 - before, after

Загальні відомості:
Застосовується для відображення бажаного контенту до/після вмісту елемента до якого він додається
css:
.text_ba {line-height: 32px;font-size: 32px;margin: 0 0 30px 0;}
.list_ba li, .list_b li {font-size: 24px;margin: 0px 0px 20px 0px;}
.list_ba li::before { content: "`";}
.list_ba li::after { content: "'";}
.list_b li::before content: ""; width: 10px; height: 10px; border-radius: 50%; background-color: red; display: inline-block; margin: 0 10px 0 0;

Приклад 4 - input -ms-clear

Загальні відомості:
Прибирає хрестик у полі вооду для IE & Edge
css:
input {width: 200px;height: 20px;border: solid 1px #000;}
input::-ms-clear { display: none;}

Приклад 5 - before + hover

Загальні відомості:
Можна комбінувати псевдокласи з псевдоелементами у певній послідовності:
спчатку псевдоклас а потім псевдоелемент
css:
.text_bh {line-height: 32px;font-size: 32px;margin: 0 0 30px 0;}
.list_bh li, .list_b li {font-size: 24px;margin: 0px 0px 20px 0px;}
.list_bh li::before content: ""; width: 10px; height: 10px; border-radius: 50%; background-color: red; display: inline-block; margin: 0 10px 0 0;
.list_bh li:hover::before { background-color: yellow; cursor: pointer;}
