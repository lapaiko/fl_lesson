Приклад 2 - Зображення у колонках

Загальні відомості:
- зображення у колонках складається з :
 - div - блоку зображення
  - img - зображення
- блоку зображення div:
  - position: relative; - позиціонування - відносне (адаптивна верстка)
  - padding: 0 0 58% 0; - задаємо висоту блоку і картинки (адаптивна верстка)
  - overflow: hidden; за межами блоку все скриваємо
- зображенню img:
  - position: absolute; - позицонування - абсолютне (адаптивна верстка)
  - top: 0; - відступ зверху =0 (адаптивна верстка)
  - width: 100%; - ширина
  - height: 100%; - висота
  - object-fit: cover; (потрбне значення) - тип заповнення
   - fill - повністю заповнює ширину та висоту;
   - contain - повністю помістись у батьківсбкий контейнер
   - none - зображення повністю вбудовуються у батьківсбкий контейнер
   - cover - зображення маштабуються так, щоб максимально вбудуватись у батьківсбкий контейнер
 - object-position:center; (потрбне значення) - позиціонування зображеня:
   - center - центрування по вертикалі/горизонталі;
   - top - вирівнювання по верхньому краю батьківського контейнеру
   - bottom - вирівнювання по нижньому краю батьківського контейнеру
   - left - вирівнювання по лівому краю батьківського контейнеру
   - right - вирівнювання по правому краю батьківського контейнеру
   - 0 0 - відступ злівого та верхнього краю батьківського контейнеру у пікселях/процентах
css:
.cards__row { display: flex; margin: 0 -15px;}
.cards__column { flex: 1 1 33.333%; padding: 0 15px;}

.cards__image {
position: relative; // - адаптація розміру
padding: 0 0 58% 0; // - чуйна висота контейнера
overflow: hidden; // - все за межами контейнеру приховано
margin: 0 0 20px 0;}

.cards__image img {
width: 100%; // - висота
height: 100%; // - ширина
object-fit: cover; // - максимальне відображення картинки
object-position: center; // - позиціонування картинки у контейнері
position: absolute; top: 0; } // - позицонування

Приклад 3 - Зображення у рядках

Загальні відомості:
- зображення у рядках складається з :
 - div - блоку зображення
  - img - зображення
- блоку зображення div:
  - position: relative; - позиціонування - відносне (адаптивна висота)
  - flex: 0 0 33.333%; - ширина 33% (адаптивна ширина)
  - min-height: 150px; - мінімальна висота на випадок відсутності/малого контенту
  - overflow: hidden; за межами блоку все скриваємо
- зображенню img:
  - position: absolute; - позицонування - абсолютне (адаптивна верстка)
  - top: 0; - відступ зверху =0 (адаптивна висота)
  - left: 0; - відступ зліва =0 (адаптивна висота)
  - width: 100%; - ширина
  - height: 100%; - висота
  - object-fit: cover; (потрбне значення) - тип заповнення
css:
.items__title { font-size: 16px; font-weight: 700; margin: 0 0 15px 0;}
.items__item { display: flex; justify-content: space-between; margin: 0 -15px 20px -15px;}

.items__image {padding: 0 15px;
flex: 0 0 33.333%; // - адаптація ширини
min-height: 150px; // - мінімальна висоти у разі відсутності /малого контенту
position: relative;} // - адаптація розміру

.items__image img {
width: 100%; // - ширина
height: 100%; // - висота
object-fit: cover; // - максимальне відображення картинки
position: absolute; // - адаптація позиціонування
top: 0; left: 0; } // - позиціонування
