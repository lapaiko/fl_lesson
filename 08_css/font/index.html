Приклад 1 - селектори, параметр, властивості, значення

Теорія:
.bloсk { } - селектор класу, що звертається до тегів з вказаним класом
div { } - селектор типу, що звертається до вказаних тегів
div.block { }- селетор типу та класу - ПРІОРИТЕНИЙ не залежний від вкзання в листі стилів
css:
.block1, .block2{ font-size: 20px; - параметр = властивість (font-size) + значення (20px) color: red; }

p class="block1"

p class="block2"

p class=""

Приклад 2 - властивість шрифта font-family - встанолює сімейство шрифта

Теорія:
За замовчуванням в браузерах встановлено font-family = "Times New Roman"
Гарачий набір: fft + Enter = font-family: "Times New Roman", Times, Baskerville, Georgia, serif;
css:
.arial { font-family: Arial, "Helvetica Neue", Helvetica, sans-serif;}

p class="" font-family = "Times New Roman"

p class="arial" font-family = "arial"

Приклад 3 - властивість шрифта font-size - встанолює розмір шрифта

Теорія:
За замовчуванням в браузерах встановлено font-family = "Times New Roman"
Гарачий набір: fz30 + Enter = font-size: 30px;
css:
.size30 { font-size: 30px; }

p class="" font-family = "Times New Roman"

p class="size30" font-size = "30px;"
