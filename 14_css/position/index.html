Приклад 1 - позиціонування position:static

Загальні відомості:
position - позіціонування елементів відносно інших елементів/вікна
static - позиціонування за властивостями типу елемента (стрічковий/блочний) static - значення за замовчуванням у всіх елементів
(стрічковий/блочний)
- вказувати не обовязково
css:
.block__static { border: 5px solid red; max-width: 800px; height: 500px; margin: 0 auto;}
.block__static div { line-height: 50px; font-size: 18px; text-align: center; font-weight: 700;}
.block__static_1 { position:static; background-color: #ece89d;}
.block__static_2 { position:static; background-color: #5e5373; color: #fff;}
.block__static_3 { position:static; background-color: #18b5a4;}

Приклад 2 - позиціонування position:relative

Загальні відомості:
relative - позиціонується за початковими координатами батьківського блоку
relative - використовується разом з властивостями: left, right, top, bottom, height, width, z-index
z-index - вказую номер слою, де чим більше значення тим вище слой (контроль елементів позиціонування за віссю z)
z-index - за замовчуванням = 0
- при встановлені значень для left, right елемент перноситься але місце під нього залишається і займає місце згідно значень height, width
css:
.block__relative { border: 5px solid red; max-width: 800px; height: 500px; margin: 0 auto;}
.block__relative div { line-height: 50px; font-size: 18px; text-align: center; font-weight: 700;}
.block__relative_1 { position:relative; background-color: #ece89d;}
.block__relative_2 { position:relative; background-color: #5e5373; color: #fff; height: 100px; top: 60px; left: 60px; z-index: 1;}
.block__relative_3 { position:relative; background-color: #18b5a4;}
block__relative_1 z-index: 0
block__relative_2 z-index: 1
block__relative_3 z-index: 0

Приклад 3 - позиціонування position:absolute

Загальні відомості:
absolute - позиціонується за початковими координати вікна браузера або батьківського блоку з позиціонуванням relative;
absolute - використовується разом з властивостями: left, right, top, bottom, height, width, z-index
z-index - за замовчуванням = 0
- місце під блок не залишається (блок виривається)!!!
- розмір блоку за замовченням приймає розмір вмісту (контенту)
- z-index - вказую номер слою, для елементів з position:absolute
- z-index - в межах батьківського елементи з position:absolute перекривають елементи з position: static не залежно від їх значення
- width, height = 100% - приймає розміри батьківського блоку/вікна браузера }
css:
.block__absolute { border: 5px solid red; max-width: 800px; height: 500px; margin: 0 auto;}
.block__absolute div { line-height: 50px; font-size: 18px; text-align: center; font-weight: 700;}
.block__absolute_1 { background-color: #ece89d; z-index: 3}
.block__absolute_2 { position:absolute; background-color: #5e5373; color: #fff; top: 60px; left: 60px;z-index: 1;}
.block__absolute_22 { position: absolute; background-color: #dda133; color: #fff; top: 25px; left: 40px; width: 100%;}
.block__absolute_3 { background-color: #18b5a4; position: relative; z-index: 2}
block__absolute_1 static z-index: 3
block__absolute_2 absolute z-index: 1
block__absolute_22 absolute z-index: 0
block__absolute_3 relativ z-index: 2

Приклад 4 - позиціонування position:fixed

Загальні відомості:
fixed - позиціонуєтьсяя за координатами вікна браузера
fixed - використовується разом з властивостями: left, right, top, bottom, height, width, z-index
z-index - за замовчуванням = 0
- розмір блоку за замовченням приймає розмір вмісту (контенту)
- width, height = 100% - приймає розміри вікна браузера
- місце під блок не залишається (блок виривається)!!!
css:
.block__fixed { border: 5px solid red; max-width: 800px; height: 500px; margin: 0 auto;}
.block__fixed div { line-height: 50px; font-size: 18px; text-align: center; font-weight: 700;}
.block__fixed_1 { background-color: #ece89d; z-index: 1}
.block__fixed_2 { position:fixed; background-color: #5e5373; color: #fff; top: 0px; left: 0px;}
.block__fixed_3 { background-color: #18b5a4;position: relative; z-index: 2}
block__fixed_1 static z-index: 1
block__fixed_2 fixed z-index: 0
block__fixed_3 relative z-index: 2

Приклад 5 - позиціонування position: sticky

Загальні відомості:
sticky - позиціонується по вікну браузера при досягненні значень top, bottom елементу під час скролу контенту
sticky - використовується разом з властивостями: left, right, top, bottom, height, width, z-index
z-index - за замовчуванням = 0
- підтримується не всіма браузерами
css:
.block__stick { border: 5px solid red; max-width: 800px; height: 500px; margin: 0 auto;}
.block__stick div { line-height: 50px; font-size: 18px; text-align: center; font-weight: 700;}
.block__stick_1 { background-color: #ece89d;}
.block__stick_2 { position:stick; background-color: #5e5373; color: #fff;}
.block__stick_3 { background-color: #18b5a4;}
.block__stick_4 { position:stick; background-color: #8a5b03;; color: #fff;}
.block__stick_5 { background-color: #0a7369;}

Приклад 6 - Складності позиціонування absolute

Загальні відомості:
- виривається з контейнера
- перкриває собою решту блоків
- постійний контроль за z-index
- складно робити верстку з динамічним контентом
css:
.block__complexities { border: 5px solid red; max-width: 800px; height: 500px; margin: 0 auto; position: relative}
.block__complexities div { line-height: 50px; font-size: 18px; text-align: center; font-weight: 700;}
.block__complexities_1 { background-color: #ece89d; position: absolute; top: 0; left: 0; width: 50%;}
.block__complexities_2 { background-color: #5e5373; ; color: #fff; position: absolute; top: 0; right: 0; width: 50%;}
.block__complexities_3 { background-color: #18b5a4; position: relative; z-index: 1;}
