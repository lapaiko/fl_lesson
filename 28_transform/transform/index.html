Трансформація об'єктів - transform - 2D

Приклад 1 - Переміщення - translate

Загальні відомості:
- transform - трансформація об'єктів:
   - блочні - трансформуються
   - інлайн - не трансформується
   - псевдоелементи - не трансформуються
   - display: inline-block; - трансформує інлайноб'єкти та псквдоелементи
- об'єкти, що трансформуються не впливають на оточуючі обʼєкти
- translate(x, y), translateX, translateY:
   - зсуває елемент на норве місце,
   - перміщуає відносно початкового положення об'єкта вправо і вниз
   - використовує координати X, Y
    - у пікселях, процентах
    - можуть бути відємними
css:
.content__block { position: relative;}
.content__transform { position: relative; padding: 10px 20px 60px 20px; margin: 0 0 20px 0; border-bottom: 1px solid #333;}
.transform__2d { display: flex; justify-content: center; align-items: center; position: relative; width: 150px; height: 150px; font-size: 14px; line-height: 20px; text-transform: uppercase; text-align: center; transition: all 0.8s ease 0s;}

.translate__0,
.translate__center_left { color: #999; border: 2px dashed #999; z-index: 2;}

.translate__50,
.translate__center_align { margin: -150px 0 0 0; color: #fff; background-color: #794f45; z-index: 1;}

.translate__animation { color: #fff; background-color: #794f45;}
.translate__animation:hover { transform: translate(20px, 20px);}

Приклад 2 - Маштабування - scale

Загальні відомості:
- scale(x, y), scaleX, scaleY:
   - X - маштабування по ширині (1 за замовчуванням)
   - Y - маштабування по висоті (1 за замовчуванням)
   - маштабує елементи (збільшує/зменшує)
   - відємне значення - відображає елемент зеркально
css:
.scale__0 { color: #999; border: 2px dashed #999;}
.scale__width, .scale__height, .scale__all, .scale__gorizont, .scale__vertical { color: #fff; transform: scale(1); background-color: #7a956b;}

.scale__width:hover { margin: 0px 0 0 75px; transform: scale(2, 1);}
.scale__height { margin: 50px 0;}
.scale__height:hover { transform: scale(1, 1.5);}
.scale__all:hover { transform: scale(0.5);}
.scale__gorizont:hover { transform: scale(-1, 1);}
.scale__vertical:hover { transform: scale(1, -1);}

Приклад 3 - Обертання - rotate

Загальні відомості:
- rotate(Xdeg):
   - повертає елемент за годиниковою стрілкою на заданий градус
   - відємний градус повертає елемент проти годинникової стрілки
   - Xdeg - формат запису значення кута + deg
css:
.rotate__0 { color: #999; border: 2px dashed #999; margin: 0px 20px;}
.rotate__45,.rotate__-90,.rotate__180,.rotate__-360 { color: #fff; background-color: #7a956b; margin: 40px 20px; transform: rotate(0);}

.rotate__45:hover { transform: rotate(45deg);}
.rotate__-90:hover { transform: rotate(-45deg);}
.rotate__180:hover { transform: rotate(180deg);}
.rotate__-360:hover { transform: rotate(-360deg);}

Приклад 4 - Деформація - skew

Загальні відомості:
- skew(Xdeg, Ydeg), skewX, skewY:
   - деформує (нахиляє) сторони об'єкта відносно координатних вісей
   - якщо вказано один кут тоді інший визначатиметься браузером автоматично
   - Xdeg, Ydeg - формат запису значення кута + deg
css:
.skew__0 { color: #999; border: 2px dashed #999; margin: 0px 20px;}

.skew__vertical:hover { transform: skew(0deg, 20deg);}
.skew__all:hover { transform: skew(-20deg, -200deg);}

Приклад 5 - Обєднання кількох трансформацій - matrix

Загальні відомості:
- matrix(a, c, b, d, x, y):
  - обєднання кількох трансформацій у одній функції
  - включає:
    - c, b - ( з маштабом a=1, d=1 ) нахил (деформація) по горизонталі та вертикалі відповідно
    - x, y - ( з маштабом a=1, d=1 ) зміна положення по горизонталі та вертикалі відповідно
    - c, b, x, y - з з маштабом a=0, d=0 поворот (сумнівна властивість!!!)
  - властивості вказуються без одиниць
css:
.matrix__0 { color: #999; border: 2px dashed #999; margin: 0px 20px;}
.matrix__mashtab,.matrix__skew,.matrix__translate { color: #fff; background-color: #77608d; margin: 40px 20px;}

.matrix__mashtab:hover { margin: 0 0 0 95px; transform: matrix(2, 0, 0, 0.7, 0, 0);}
.matrix__skew:hover { transform: matrix(1, 0.3, 0.2, 1, 0, 0);}
.matrix__translate:hover { transform: matrix(1, 0, 0, 1, 70, 45);}

Приклад 7 - Зміщення центру трансформації - transform-origin

Загальні відомості:

- transform-origin - зміщає центр трансформації відносно якого виконується трансформація та приймає значення:
  - top - з верху
  - left - з ліва
  - right -з права
  - bootom - з верху
  - 0 - значення задано числом
  - у псевдокласах :hover не задавати зміщення транформації
css:
.transform-origin__0 { color: #999; border: 2px dashed #999; margin: 0px 20px;}
.transform-origin__center,.transform-origin__top-right {color: #fff;background-color: #794f45;}
.transform-origin__center {margin: 40px 20px;transform-origin: center;}
.transform-origin__center:hover {transform: rotate(45deg);}
.transform-origin__top-right{margin: 20px 20px 60px 20px;transform-origin: top right;}
.transform-origin__top-right:hover {transform: rotate(-45deg);}

Приклад 8 - Перспектива - глибина та точка відліку - perspective, perspective-origin

Загальні відомості:

  - d/(d-z)
  - perspective-origin - встановлюють точку відліку координат
css:
.content-3d__wrapper { width: 150px; height: 150px; background-color: rgba(0, 0, 0, 0.2); perspective-origin: center; perspective: 400px;}
.content-3d { display: flex; justify-content: center; align-items: center; text-align: center; text-transform: uppercase; font-size: 16px; width: 150px; height: 150px; transition: all 0.8s ease 0s;}

.content-3d__perspective { background-color: #7a956b;}
.content-3d__perspective:hover { transform: rotateX(30deg);}

Приклад 14 - Стиль трансформації - transform-style
