Приклад 1 - display:flex

Загальні відомості:
display:flex - блочний flex-контенер за замовчуванням:
- контейнер займає всю ширину області в якій він знаходиться (блочний)
- елементи стають в ряд, не перходять на іншу стрічку і не стискаються (строчний)
- flex-box - flex-conteiner - flex-element - набір властивостей, що дозволяє задавати висоту, ширину, простір навколо та порядок елементів
css:
//флекс контейнер
.blokflex__row{
display: flex;
.border: 20px solid #ece89d; margin: 0 0 10px 0;
}
//флекс елемент
.blokflex__column { border: 20px solid #5e5373; }
//вміст
.blokflex__item { background-color: #18b5a4; padding: 50px; font-size: 50px; color: #fff; font-weight: 700;}

Приклад 2 - display:inline-flex

Загальні відомості:
display:inline-flex - строчний flex-контенер за замовчуванням:
- контейнер займає ширину вмісту (строчний)
- елементи стають в ряд, не перходять на іншу стрічку і не стискаються (строчний)
- flex-box - flex-conteiner - flex-element - набір властивостей, що дозволяє задавати висоту, ширину, простір навколо та порядок елементів
css:
//батьківський блок
.blokinline-flex { text-align: center; //центрування вмісту блоку
//флекс контейнер
.blokinline-flex__row {

//вміст

Приклад 3 - justify-content - вирівнювання за основною віссю

Загальні відомості:
justify-content - вирівнювання за основною віссю:
- flex-start - вирівнювання по лівому краю (між елементами немає простору);
- center - вирівнювання по центру (між елементами немає простору);
- flex-end - вирівнювання по правому краю (між елементами немає простору)
- space-between - простір між крайням контейнера і елементом відсутній, а між елементами однаковий (відступів немає);
- space-around - простір між крайням контейнера і елементом складає половину простору між елементами, а між елементами однаковий (відступів немає);
- властивість - flex-element
css:
//флекс контейнер
.blokjustify-content__row{
display: flex;
&_left { justify-content: flex-start; }
&_center { justify-content: center; }
&_right { justify-content: flex-end; }
&_between { justify-content: space-between; }
&_around { justify-content: space-around; }
.border: 20px solid #ece89d; margin: 0 0 10px 0;
}
//флекс елемент
.blokjustify-content__column { border: 20px solid #5e5373; }
//вміст
.blokjustify-content__item { background-color: #18b5a4; padding: 50px; font-size: 50px; color: #fff; font-weight: 700;}

justify-content: flex-start; (за замовчуванням)

Приклад 4 - align-items - вирівнювання за не основною віссю

Загальні відомості:
align-items - вирівнювання за не основною віссю: - stretch - елементи вирівннюються по верху, а висота приймає значення елемента з найвищим контентом; - flex-start - елементи вирівннюється по верху, а висота приймає значення контенту елементу; - flex-end - елементи вирівннюються по низу, а висота приймає значення контенту елементу; - flex-center - елементи вирівннюються по середині, а висота приймає значення контенту елементу; - baseline - елементи вирівннюються по базовій лінії, а висота приймає значення контенту елементу;
- властивість - flex-element
css:
//флекс контейнер
.blokalign-items__row{
display: flex;
&_stretch { align-items: stretch;}
&_start { align-items: flex-start;}
&_end { align-items: flex-end;}
&_center { align-items: center;}
&_baseline { align-items: baseline;}
.border: 20px solid #ece89d; margin: 0 0 10px 0;
}
//флекс елемент
.blokalign-items__column { border: 20px solid #5e5373; }
//вміст
.blokalign-items__item { background-color: #18b5a4; padding: 50px; font-size: 50px; color: #fff; font-weight: 700;}

align-items: stretch; (за замовчуванням)

Приклад 5 - flex-wrap - перехід елемента на нову стрічку

Загальні відомості:
flex-wrap - керує перехідом елементу на нову стрічку: - nowrap - елементи не переходять на нову стрічку; - wrap - елементи переходять на нову стрічку; - wrap-reverse - елементи переходять на нову стрічку у зворотньому порядку.
- властивість - flex-element
css:
//флекс контейнер
.blokwrap__row{
display: flex;
&_nowrap {flex-wrap: nowrap;}
&_wrap { flex-wrap: wrap;}
&_wrap-reverse { flex-wrap: wrap-reverse;}
.border: 20px solid #ece89d; margin: 0 0 10px 0;
}
//флекс елемент
.blokwrap__column { border: 20px solid #5e5373; }
//вміст
.blokwrap__item { background-color: #18b5a4; padding: 150px; font-size: 50px; color: #fff; font-weight:1 700;}
