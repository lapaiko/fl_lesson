Приклад 1 - align-self

Загальні відомості:
align-self - перевизначає вирівннювання елементів:
- stretch - елементи вирівннюються по верху, а висота приймає значення елемента з найвищим контентом;
- center - елементи вирівннюються по середині, а висота приймає значення контенту елементу;
- flex-start - елементи вирівннюється по верху, а висота приймає значення контенту елементу;
- flex-end - елементи вирівннюються по низу, а висота приймає значення контенту елементу;
- має тіж властивості що align-items, але застосовується лише до flex-елементу
- перевстановлює вирівнювання для кожного елементу окремо
css:
//флекс контейнер
.blok-align-self__row{
display: flex;
align-items: center;
.border: 20px solid #ece89d; margin: 0 0 10px 0;
}
//флекс елемент
.blok-align-self__column { border: 20px solid #5e5373;
&_stretch { align-self: stretch;}
&_center { align-self: center;} &_start { align-self: flex-start;}
&_end { align-self: flex-end;}
}
//вміст
.blok-align-self__item { background-color: #18b5a4; padding: 50px; font-size: 50px; color: #fff; font-weight: 700;}

align-self: stretch; (за замовчуванням)

Приклад 2 - order - порядок виводу flex-елементів

Загальні відомості:
order - змінює порядок виводу flex-елементів
- змінює порядок виводу flex-елементів - номер ставиться для всіх flex-елементів (навіть для тих хто його не змінив)
css:
//флекс контейнер
.blok-order__row{
display: flex;
.border: 20px solid #ece89d; margin: 0 0 10px 0;
}
//флекс елемент
.blok-order__column {
&_1 { order: 3;}
&_2 { order: 2;}
&_3 { order: 1; }
border: 20px solid #5e5373; }
//вміст
.blok-order__item { background-color: #18b5a4; padding: 50px; font-size: 50px; color: #fff; font-weight: 700;}

order: порядковий номер

Приклад 3 - flex-basis - базовий розмір flex-елементів

Загальні відомості:
flex-basis - базовий розмір flex-елементів: - auto - базовий розмір flex-елементу рівний розміру контенту (значення за замовчуванням); - __px - базовий розмір flex-елементу у пікселях; - __% - базовий розмір flex-елементу у відсотках;
- встановлює базовий розмір flex-елементу
css:
//флекс контейнер
.blok-basic__row{
display: flex;
.border: 20px solid #ece89d; margin: 0 0 10px 0;
}
//флекс елемент
.blok-basic__column {
&_auto { flex-basis: auto;}
&_p30 { flex-basis: 30%;}
&_x250 { flex-basis: 250px;}
border: 20px solid #5e5373; }
//вміст
.blok-basic__item { background-color: #18b5a4; padding: 50px; font-size: 50px; color: #fff; font-weight: 700;}

flex-basis: auto; (за замовчуванням)

Приклад 4 - flex-grow - можливість flex-елементів ставати більшим за значення flex-basis

Загальні відомості:
flex-grow - можливість flex-елементів ставати БІЛЬШИМ за значення flex-basis:
- 0 - заборонено ставати більшим (за замовчуванням);
- 1 - дозволено ставати більшим;
- встановлює можливість flex-елементів ставати більшим за значення flex-basis
css:
//флекс контейнер
.blok-grow__row{
display: flex;
.border: 20px solid #ece89d; margin: 0 0 10px 0;
}
//флекс елемент
.blok-grow__column {
&_0{ flex-basis: 250px;; flex-grow: 0; }
&_1{ flex-basis: 250px; flex-grow: 1; }
border: 20px solid #5e5373; }
//вміст
.blok-grow__item { background-color: #18b5a4; padding: 50px; font-size: 50px; color: #fff; font-weight: 700;}

flex-grow: 0; (за замовчуванням) flex-basis: 250px;

Приклад 5 - flex-shrink - можливість flex-елементів ставати меншими за значення flex-basis

Загальні відомості:
lex-shrink - можливість flex-елементів ставати МЕНШИМ за значення flex-basis:
- 0 - заборонено ставати більшим;
- 1 - дозволено ставати більшим (за замовчуванням);
- *
css:
//флекс контейнер

.border: 20px solid #ece89d; margin: 0 0 10px 0;
}
//флекс елемент
.blok-shrink__column { border: 20px solid #5e5373; }
//вміст
.blok-shrink__item { background-color: #18b5a4; padding: 50px; font-size: 50px; color: #fff; font-weight: 700;}

flex-shrink: 1; (за замовчуванням) flex-basis: 250px;

Приклад 6 - flex - встановлює значення flex-grow flex-shrink flex-basis

Загальні відомості:
flex - встановлює значення flex-grow flex-shrink flex-basis у наведеному порядку
- компактний запис flex-grow flex-shrink flex-basis
css:
//флекс контейнер
.blok-flex__row{
display: flex;
.border: 20px solid #ece89d; margin: 0 0 10px 0;
}
//флекс елемент
.blok-flex__column {
flex: 0 1 auto;
border: 20px solid #5e5373; }
//вміст
.blok-flex__item { background-color: #18b5a4; padding: 50px; font-size: 50px; color: #fff; font-weight: 700;}

flex:0 1 auto; (за замовчуванням)
