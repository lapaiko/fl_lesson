Приклад 1 - flex-direction - встановлює основну вісь

Загальні відомості:
flex-direction - встановлює основну вісь для flex-елементів:
- row - вистроювання flex-елементів у ряд з ліва направо (за замовчуванням);
- row-reverse - вистроювання flex-елементів у ряд з права на ліво у зворотньому порядку;
- column - вистроювання flex-елементів у колонку;
- column-reverse - вистроювання flex-елементів у колонку у зворотньому порядку;
- за основною віссю вистроюються flex-елементи
css:
//флекс контейнер
.blodirection__row{
display: flex;
&_row{ flex-direction: row; }
&_row-reverse { flex-direction: row-reverse;}
&_column-stretch { flex-direction: column; align-items: stretch;}
&_column-flex-start { flex-direction: column; align-items: flex-start;}
&_column-flex-end { flex-direction: column; align-items: flex-end;}
&_column-space-between { flex-direction: column; justify-content: space-between; height: 800px;}
&_column-maket { flex-direction: column; height: 800px;}
.border: 20px solid #ece89d; margin: 0 0 10px 0;
}
//флекс елемент
.blodirection__column { border: 20px solid #5e5373;
&_m2{ flex: 1 1 auto; }
}
//вміст
.blodirection__item { background-color: #18b5a4; padding: 50px; font-size: 50px; color: #fff; font-weight: 700;}
