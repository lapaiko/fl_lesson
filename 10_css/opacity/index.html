Приклад 2 - border-radius встановлює радіус закруглення границі

Теорія: border-radius: 5px; встановлює радіус закруглення границі
за замовчуваням border-radius: 0px;
border-radius: 10px 10px 10px 10px; - встановлення радіусів з верхнього лівого кута і далі радіуси кутів за годинниковою стрілкою
Гарачий набір:
btrr + Enter = border-top-right-radius: ;
btlr + Enter = border-top-left-radius: ;
bblr + Enter = border-bottom-left-radius: ;
bbrrr + Enter = border-bottom-right-radius: ;
css:
.example-block-border-radius { border-radius: 5px; }
.example-block-border-radius-lrbt { border-top-right-radius: 10px; border-top-left-radius: 20px; border-bottom-left-radius: 15px; border-bottom-right-radius: 25px; }
.example-block-border-radius-all { overflow: hidden; border-radius: 50px 40px 40px 0px; }
.example-block-border-radius-circle { overflow: hidden; width: 100px; height: 100px; border-radius: 50%; }
.example-string-border-radius { border-radius: 5px; }
.example-img-border-radius { border-radius: 5px; }
