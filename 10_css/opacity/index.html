Приклад 5 - opacity визначає рівень прозорості елемента

Теорія:
працює як блочними так і строчними елементами
opacity: 1; - за замовчуванням де 1 - не прозорий, 0 - прозорий
opacity: 0; - (на відміну від display: none;) залишає місце під елемент роблячи його прозорим
opacity: 0; - (на відміну від display: none;) можна взаємодіяти з елементами такими як: a, input
opacity: 0; - (на відміну від display: none;) можна робити анімацію елементів
Гарачий набір:
op0.5 + Enter = opacity: 0.5
css:
.example-block-opacity { opacity: 0.3; }
.example-string-opacity { opacity: 0.3; }
.example-img-opacity { opacity: 0.3; }

Приклад 6 - visibility призначена для відображення або скриття блоку

Теорія:
працює як блочними так і строчними елементами
visibility: visible; - за замовчуванням
visibility: hidden; - (на відміну від display: none;) як і opacity: 0; залишає місце під елемент роблячи його прозорим
visibility: hidden; - (на відміну від opacity: 0;) як і в display: none; не можна взаємодіяти з елементами
Гарачий набір:
v + Enter = visibility: hidden;
css:
.example-block-visibility-h { visibility: hidden; }
.example-string-visibility-h { visibility: hidden; }
.example-img-visibility-h { visibility: hidden; }
