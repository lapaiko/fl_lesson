Приклад 1 - hover

Загальні відомості:
- спрацьовує при наведені на елемент - використовується всіх елементів

:hover

- наведення на ссилку

div::hover

- наведення на блок
css:

.link:hover {text-decoration: underline; color: blue; }

.block:hover { background: blue; }

.textbutton:hover { background: #333; color: yellow; }

Приклад 5 - first-child, last-child

Загальні відомості:
- дозволяє звернутись до першрго або останнього елементу у блоці

:first-child

- звертається до першого елементу у блоці

last-child

- звертається до останнього елементу у блоці
css:

.list li:first-child { color: green; }

.list li:last-child { color: greenyellow; }

.block-first-last p:first-child { text-transform: uppercase; }

.block-first-last p:last-child { text-transform: uppercase; }

first-child - не працює томущо тег < р> не є першим

Приклад 6 - nth-child

Загальні відомості:
- звернення до елементу за його номером у блоці

:nth-child(3)

- звертається до третього елементу у блоці

nth-child(odd)

- звертається до всіх елементів з непарним номером

nth-child(even)

- звертається до всіх елементів з непарним номером
css:

.block-nth p:nth-child(4) { text-transform: uppercase; }

.block-nth p:nth-child(1) { text-transform: uppercase; }

.block-nth p:nth-child(odd) { background: yellow; }

.block-nth p:nth-child(even) { background: #eee; }

.list-ntt li:nth-child(4) { text-decoration: underline; }
