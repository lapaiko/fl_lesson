Приклад 1 - Тривалість переходу - transition-duration

Загальні відомості: - transition-duration - тривалість переходу
- transition-duration: 0.5s; - 0,5 секунди
- transition-duration: 500ms; - 500 мілісекунд
- transition-duration: 5s; - 5 секунд
- transition-duration: 3s, 200ms, 0.8s; - 3 секунди тривалість властивості-1; 200 мілісекунд тривалість властивості-2; 0,8 секунди тривалість властивості-3
css:
.button { position: relative; top: 0 display: inline-block; line-height: 60px; align-items: center padding: 0 30px text-decoration: none; text-transform: uppercase; letter-spacing: 2px; font-size: 18px border-radius: 10px;}
.button__duration { color: #fff; background-color: #5e5373; box-shadow: 0 5px 0px #3c354a; transition-duration: 500ms;}
.button__duration:hover { top: 2px; background-color: #18b5a4; box-shadow: 0 3px 0px #3c354a;}

Приклад 2 - Перелік властивостей переходу - transition-property

Загальні відомості: - transition-property - вказано перелік властивостей за якими виконується перехід
- transition-property: all; - для всіх властивостей перходу
- transition-property: background-color; - перхід тільки для фону
- transition-property: background-color, top; - перхід для фону, відступу зверху
css:
.button { position: relative; top: 0 display: inline-block; line-height: 60px; align-items: center padding: 0 30px text-decoration: none; text-transform: uppercase; letter-spacing: 2px; font-size: 18px border-radius: 10px;}
.button__property { color: #fff; background-color: #5e5373; box-shadow: 0 5px 0px #3c354a; transition-duration: 500ms, 1500ms; transition-property: background-color, top;}
.button__property:hover { top: 2px; background-color: #18b5a4; box-shadow: 0 3px 0px #3c354a;}

Приклад 3 - Затримка переходу - transition-delay

Загальні відомості:
- transition-delay - затримка виконання переходу
- transition-delay: 200ms; - затримка виконання на 200мс
- transition-duration: 0s, 100ms, 100ms; - 0c затримка 1-перходу; 100мc затримка 2-перходу;100мc затримка 3-перходу
css:
.button { position: relative; top: 0 display: inline-block; line-height: 60px; align-items: center padding: 0 30px text-decoration: none; text-transform: uppercase; letter-spacing: 2px; font-size: 18px border-radius: 10px;}
.button__delay { color: #fff; background-color: #5e5373; box-shadow: 0 5px 0px #3c354a; transition-delay: 0s, 500ms, 500ms; transition-duration: 0s, 100ms, 100ms; transition-property: background-color, top, box-shadow;}
.button__delay:hover { top: 2px; background-color: #18b5a4; box-shadow: 0 3px 0px #3c354a;}

Приклад 4 - Стиль/сценарій переходу - transition-timing-function

Загальні відомості:
- transition-timing-function
- ease - (за замовчуванням) початок-пришвидшується, середина-уповільнюється, завершення-пришвидшується
- easy-in - початок-повільний, середина-рівна, завершення-пришвидшується
- easy-out - початок-швидкий, середина-рівна, завершення-уповільнюється
- easy-in-out - початок-повільний, середина-рівна, завершення-уповільнюється
- linear - початок, середина, завершення - -рівномірні
- steps(5, start) - тривалість перходу поділена на 5 кроків, анімація починається відразу на кожному кроці
- steps(5, end) - тривалість перходу поділена на 5 кроків, анімація починається з атримкою на кожному кроці
- cubic-bezier - створення власного сценарію

.button__timing-function {position: relative;top: 0display: block;width: 300px;line-height: 60px;align-items: center; padding: 0 30px; margin: 20px 0 0 0; text-decoration: none; text-transform: uppercase; letter-spacing: 2px; font-size: 18px; border-radius: 10px; cursor: pointer; color: #fff; transition-property: width; transition-duration: 1s;}
.button__timing-function_easy { background-color: #5e5373; transition-timing-function: ease;}

.button__timing-function_linear { background-color: #18b5a4; transition-timing-function: linear;}
.button__timing-function_steps-start { background-color: #ece89d; transition-timing-function: steps(5, start);}

Приклад 5 - Тривалість переходу - transition

Загальні відомості:
- transition: property (властивість), duration (тривалість), timing-function(сценарій перходу), delay (затримка) - анімація встановлюється як на класі так і псевдокласі
- transition: all 1s ease 0s; - всі властивості, тривалість 1с, ease-сценарій, затримка=0
- transition: padding 1s ease 0s, color 2s ease-in 0.5s; - padding, тривалість 1с, ease-сценарій, затримка=0 та color, тривалість 2с, ease-in-сценарій, затримка 0.5c
css:
.button { position: relative; top: 0 display: inline-block; line-height: 60px; align-items: center; padding: 0 30px text-decoration: none; text-transform: uppercase; letter-spacing: 2px; font-size: 18px border-radius:10px;}
.button__transition { color: #fff; background-color: #5e5373; box-shadow: 0 5px 0px #3c354a; transition: all 1s ease 0s; /*transition: padding 1s ease 0s, color 2s ease-in 0.5s;*/}
.button__transition:hover { top: 2px; background-color: #18b5a4; box-shadow: 0 3px 0px #3c354a; transition: all 2s ease 0s;}
