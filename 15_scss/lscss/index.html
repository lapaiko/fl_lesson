Приклад 5 - Змінні та імпорт

Загальні відомості:
"$" - префікс змінних
@import - ключове слово за яким у лапках вказується назва scss файлу для імпорту (наприклад "nullstylesass_15.scss");
- приклад взастосування змінних
$fz: 100px;
block__val_text { font-size: $fz;}
- приклад імпорту scss нульового стилю
@import "nullstylesass_15.scss";
- при редагуванні імпортованого scss файлу зміни відбуваються у основному файлі
css:
$fz: 100px;
$color_1: #000;
$color_2: #fff;
.block__val {
padding: 30px;line-height: 60px;
.block__val_text { font-size: $fz;
span { color: green;}
}
}
.link__val {color: $color_1;text-decoration: underline;font-size: $fz;}
.block__val & { font-size: $fz;}
&:hover { color: $color_2;}
}

Приклад 6 - шаблони

Загальні відомості:
"%" - префікс шаблону "." - префікс стилю, що млже бути використаний як шаблон "@extend" - виклик шаблону
- порядок розташування шаблону має значення!!!
- шаблон, з що ніде не використовується в основному css:
-- з префіксом "%" виводитись не буде
-- з префіксом "." виводитись буде
css:
$fz: 100px;
$color_1: #000;
$color_2: #fff;
.tplbordergreen { font-size: 100px; border-bottom: 5px dashed green;}
.block__shab {
padding: 30px;line-height: 60px;
.block__shab_text { font-size: $fz;
span { color: green;}
}
}
.link__shab {color: $color_1;text-decoration: underline;font-size: $fz;
.block__shab & { font-size: $fz;}
&:hover { color: $color_2;}
@extend .tplborder;
}
.link__shab-2 {
@extend .tplborder; font-size: 80px;
}
%tplborder { font-size: 100px; border-bottom: 5px dashed red;}

Приклад 7 - Міксіни

Загальні відомості:
@mixin назва(змінні) { параметри }- аналог процедури у якій перлічені властивості з значеннями у вигляді змінних
@include - виклик міксіна
- кількість змінних не обмежена
- можливо застосовувати математичні операції внутрі міксіна
css:
$fz: 100px;
$color_1: #000;
$color_2: #fff;
@mixin fontz($f, $c) {
font-size: $f;
color: $c;
}
.block__mix {padding: 30px;line-height: 60px;
.block__mix_text {font-size: $fz;
span { color: green; }
} }
.link__mix { color: $color_1; text-decoration: underline; font-size: $fz;
.block__mix & { font-size: $fz;}
&:hover { color: $color_2;}
}
.link__mix-2 {
@include fontz(50px, green);
border-bottom: 5px dashed #000;
}
