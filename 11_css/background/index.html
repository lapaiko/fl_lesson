CSS background

Шпаргалка (методичка) CSS властивості background. «Фрилансер по жизни»

Приклад 1 - background-color

Теорія:
background керує фоном елемента, що встановлює:
- колір фону блочного елемента
- градієнтне - фонове зображення - позицію фонового зображення - задати метод прокрутки фонового зображення - маштабувати фонове зображення згідно потрібних розмірів background-color: #fff; - за замовчуванням background-color: yellowgreen; та background: yellowgreen; - рівнозначні записи
Гарачий набір:
bc + Tab = background-color: #fff;
css:
.block-background-color { background-color: yellowgreen; }
