CSS background

Шпаргалка (методичка) CSS властивості background. «Фрилансер по жизни»

Приклад 1 - background або background-color

Загальні відомості:
background керує фоном елемента, що встановлює:
- колір фону блочного елемента
- градієнтне
- фонове зображення
- позицію фонового зображення
- задати метод прокрутки фонового зображення
- маштабувати фонове зображення згідно потрібних розмірів
Примітка:
- background-color: #fff; - за замовчуванням
- background-color: yellowgreen; та background: yellowgreen; - рівнозначні записи
Гарачий набір:
- bc + Tab = background-color: #fff;
Коонвертація з HEX в RGBA

transparent

- фон прозорим,
- місце під елемент залишається,
- елемент активний
css:

.block-background-color-transparent { background: transparent; }

block-background-color-transparent

#18b5a4;

- встановлює 16-річний колір фону
css:

.block-background-color { background: #18b5a4; }

block-background-color

rgb(24, 181, 164);

- встановлює rgb - колір фону
css:

.block-background-color-rgb { background: rgb(24, 181, 164); }

block-background-color-rgb

rgb(24, 181, 164);

- встановлює rgb - колір фону з альфа-канлом (встановлює рівень прозорості від 0 до 1)
css:

.block-background-color-rgba { background: rgba(24, 181, 164, 0.5);} де , 0.5 - прозорість

block-background-color-rgba

Приклад 2 - background або background-image

Загальні відомості:
- встановлює фонове зображення елемента або задає градієнтну заливку
Гарачий набір:
- bc + Tab = background-color: #fff;
Інструмент створення градієнту

url('img/avatar.jpg');

- встановлює зображення фону елемента
css:

.block-background-image { background: url('img/avatar.jpg'); }

block-background-image

rgb(236, 232, 157);

- градієнтна заливка для старих браузерів - не працює

-moz-linear-gradient

- градієнтна заливка фона для FF3.6-15

-webkit-linear-gradient

- градієнтна заливка фона для Chrome10-25,Safari5.1-6

linear-gradient

- градієнтна заливка фона для W3C, IE10+, FF16+, Chrome26+, Opera12+, Safari7+
css: .block-background-gradient-liner { /* Old browsers */

background: rgb(236, 232, 157);

/* FF3.6-15 */

background: -moz-linear-gradient(top, rgba(236, 232, 157, 1) 0%, rgba(94, 83, 115, 1) 50%, rgba(24, 181, 164, 1) 100%);

/* Chrome10-25,Safari5.1-6 */

background: -webkit-linear-gradient(top, rgba(236, 232, 157, 1) 0%, rgba(94, 83, 115, 1) 50%, rgba(24, 181, 164, 1) 100%);

/* W3C, IE10+, FF16+, Chrome26+, Opera12+, Safari7+ */

background: linear-gradient(to bottom, rgba(236, 232, 157, 1) 0%, rgba(94, 83, 115, 1) 50%, rgba(24, 181, 164, 1) 100%);}

block-background-gradient-liner

rgb(236, 232, 157);

- градієнтна заливка для старих браузерів - не працює

-moz-radial-gradient

- градієнтна заливка фона для FF3.6-15

-webkit-radial-gradient

- градієнтна заливка фона для Chrome10-25,Safari5.1-6

radial-gradient

- градієнтна заливка фона для W3C, IE10+, FF16+, Chrome26+, Opera12+, Safari7+
css: .block-background-gradient-radial {
/* Old browsers */

background: rgb(236, 232, 157);

/* FF3.6-15 */

background: -moz-radial-gradient(center, ellipse cover, rgba(236, 232, 157, 1) 0%, rgba(94, 83, 115, 1) 50%, rgba(24, 181, 164, 1) 100%);

/* Chrome10-25,Safari5.1-6 */

background: -webkit-radial-gradient(center, ellipse cover, rgba(236, 232, 157, 1) 0%, rgba(94, 83, 115, 1) 50%, rgba(24, 181, 164, 1) 100%) ;

/* W3C, IE10+, FF16+, Chrome26+, Opera12+, Safari7+ */

background: radial-gradient(ellipse at center, rgba(236, 232, 157, 1) 0%, rgba(94, 83, 115, 1) 50%, rgba(24, 181, 164, 1) 100%) ;

}

block-background-gradient-radial

Приклад 3 - background або background-repeat

Загальні відомості:
- визначає як буде повторюватись фонове зображення
Гарачий набір:
- bc + Tab = background-color: #fff;

repeat

- фонове зображення повторюється по горионталі та вертикалі
css:

.block-background-repeat-default{ background: url('../img/avatar.jpg') repeat; }

block-background-repeat-default

no-repeat

- фонове зображення не повторюється
css:

.block-background-repeat-no{ background: url('../img/avatar.jpg') no-repeat; }

block-background-repeat-no

repeat-x

- фонове зображення повторюється по осі Х
css:

.block-background-repeat-x{ background: url('../img/avatar.jpg') repeat-x; }

block-background-repeat-x

repeat-y

- фонове зображення повторюється по осі Y
css:

.block-background-repeat-y{ background: url('../img/avatar.jpg') repeat-y; }

block-background-repeat-y

space

- фонове зображення - повторються так, щоб заповнити весь простір, а решта місця між ними залишаться порожнім
css:

.block-background-repeat-space{ background: url('../img/avatar.jpg') space; }

block-background-repeat-space

round

- фонове зображення - повторються так, щоб заповнити весь простір при цьому не рівномірно масштабуються для заповненн порожнього простору
css:

.block-background-repeat-round{ background: url('../img/avatar.jpg') round; }

block-background-repeat-round

Приклад 4 - background або background-position

Загальні відомості:
- задає початкову положення фонового зображення
Гарачий набір:
- bc + Tab = background-color: #fff;

background-position: 0 0;

- відступ по горизонталі = 0 і відступ по вертикалі =0
css:

.block-background-position-default{ background: url('../img/avatar.jpg') 0 0 no-repeat; }

block-background-position-default

background-position: 50%;

- по центру
css:

.block-background-position-center{ background: url('../img/avatar.jpg') 50% no-repeat; }

block-background-position-center

background-position: 0 50%;

- зліва
css:

.block-background-position-left{ background: url('../img/avatar.jpg') 0 50% no-repeat; }

block-background-position-left

background-position: 100% 100%;

- в нижньому правому вуглі
css:

.block-background-position-corner{ background: url('../img/avatar.jpg') 100% 100% no-repeat; }

block-background-position-corner

background-position: 25px 30px;

- відступ по горизонталі = 30px і відступ по вертикалі =20px
css:

.block-background-position-place{ background: url('../img/avatar.jpg') 25px 30px no-repeat; }

block-background-position-place

Приклад 5 - background або background-attachment

Загальні відомості:
- визначає чи буде фонове зображення прокручуватись разом з змістом елемента

scroll

- дозволяє перміщатись фону разом з вікном браузера
css:

.block-background-attachment-scroll {background: url('../img/bg.jpg') 0 0 no-repeat scroll; }

block-background-attachment-scroll

fixed

- фонове зображення стає нерухомим та встанлює його положення відносно вікна браузера
css:

.block-background-attachment-fixed {background: url('../img/bg.jpg') 0 0 no-repeat fixed; }

block-background-attachment-fixed

local

- фон фіксується з врахуванням поведінки елементу. Якщо елемент має прокрутку, то фон буде прокручуватись разом із змістом, але фон, що виходить за рамки елементу залишається на місці і не покриває зміст
css:

.block-background-attachment-local {background: url('../img/bg.jpg') 0 0 no-repeat local; }

block-background-attachment-local

Приклад 6 - background або background-size

Загальні відомості:
- маштабує фонове зображення за вказаними розмірами
- 0 0/100% 100% no-repeat - перше 100% - встановлю ширину в %/px, а другі 100% - встановлють висоту в %/px;

/ auto auto

- за замовчуванням
css:

.block-background-size-default {background: url('../img/bg.jpg')0 0/auto auto no-repeat; }

block-background-attachment-size-default

/ 100% auto

- встановлює ширину картинки
css:

.block-background-size-width {background: url('../img/bg.jpg') 0 0/100% auto no-repeat; }

block-background-attachment-size-width

/ auto 100%

- встановлює висоту картинки
css:

.block-background-size-height {background: url('../img/bg.jpg') 0 0/auto 100% no-repeat; }

block-background-attachment-size-height

/ 100% 100%

- розтягує картинку фону на всю ширину та довжину блоку
css:

.block-background-size-strech {background: url('../img/bg.jpg') 0 0/100% 100% no-repeat; }

block-background-attachment-size-strech

/ contain

- маштабує зображення з ЗБЕРЕЖЕННЯМ ПРОПОРЦІЙ по МЕНШІЙ стороні КАРТИНКИ - ширина або висота КАРТИНИ рівна ширині або висоті БЛОКУ
css: .block-background-size-contain {background: url('../img/bg.jpg') 0 0/contain no-repeat; }

block-background-attachment-size-contain

/ cover

- маштабує картинку з ЗБЕРЕЖЕННЯМ ПРОПОРЦІЙ по БІЛЬШІЙ стороні КАРТИНКИ - ширина або висота КАРТИНИ рівна ширині або висоті БЛОКУ
css:

.block-background-size-cover {background: url('../img/bg.jpg') 50%/cover no-repeat; }

block-background-size-cover

/ 200px 150px

- встановлює ширину і висоту фонової картинки. ОДНЕ значення встановлює ширину картинки а висота приймає значення auto прицьому зберігаються ПРОПОРЦІЇ КАРТИНКИ.
css:

.block-background-size-width-height {background: url('../img/bg.jpg') 0 0/200px 150px no-repeat; }

block-background-size-width-height

Приклад 7 - установка кількох фонових зображень

Загальні відомості:
- властивості background дозволяють задати кілька фонових зображень одному блоку
- кожне зображення матиме свої налаштування позиціонування, маштабуваня та прокрутки
- фонові зображення вказуються через кому

#18b5a4 url('../img/avatar.jpg') no-repeat

- колір фону + одна картинка без повтору
css:

.block-background-img-color {background: #18b5a4 url('../img/avatar.jpg') no-repeat }

block-background-img-color

url('../img/avatar_3.jpg') left no-repeat, url('../img/bg.jpg') 50%/cover no-repeat;

- дві картинки фону
css:

.block-background-img-img { background: url('../img/avatar_3.jpg') left no-repeat, url('../img/bg.jpg') 50%/cover no-repeat; }

block-background-img-img

/ contain

- три картинки фону
css:

.block-background-img-img-img { background: url('../img/avatar_2.jpg') right/50px no-repeat, url('../img/avatar_3.jpg') left no-repeat, url('../img/bg.jpg') 50%/cover no-repeat; }

block-background-img-img-img

/ cover

- чотири картинки фону
css:

.block-background-img-img-img-img { background: url('../img/avatar.jpg') 0 0/50px no-repeat, url('../img/avatar_2.jpg') 100% 100%/50px no-repeat, url('../img/avatar_3.jpg') 50% no-repeat, url('../img/bg.jpg') 50%/cover no-repeat; }

block-background-img-img-img-img
