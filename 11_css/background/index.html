Приклад 1 - background або background-color - визначає колір фону елекмента

Теорія:
background керує фоном елемента, що встановлює:
- колір фону блочного елемента
- градієнтне - фонове зображення - позицію фонового зображення - задати метод прокрутки фонового зображення - маштабувати фонове зображення згідно потрібних розмірів background-color: #fff; - за замовчуванням background-color: yellowgreen; та background: yellowgreen; - рівнозначні записи
Гарачий набір:
bc + Tab = background-color: #fff;
css:
.block-background-color-transparent { background: #18b5a4; background: transparent; }
.block-background-color { background: #18b5a4; }
.block-background-color-rgb { background: rgb(24, 181, 164); }
.block-background-color-rgba { background: rgba(24, 181, 164, 0.5); } де , 0.5 - прозорість
Коонвертація з HEX в RGBA
block-background-color-transparent
block-background-color
block-background-color-rgb
block-background-color-rgba

Приклад 3 - background або Background-repeat - визначає як буде повторюватись фонове зображення заливку

Теорія:
- repeat - фонове зображення повторюється по горионталі та вертикалі - no-repeat - фонове зображення не повторюється - repeat-x - фонове зображення повторюється по осі Х - repeat-y - фонове зображення повторюється по осі Y - space - фонове зображення - повторються так, щоб заповнити весь простір, а решта місця між ними залишаться порожнім - round - фонове зображення - повторються так, щоб заповнити весь простір при цьому не рівномірно масштабуються для заповненн порожнього простору Гарачий набір:
bc + Tab = background-color: #fff;
css:
.block-background-repeat-default{ background: url('../img/avatar.jpg') repeat; }
.block-background-repeat-no{ background: url('../img/avatar.jpg') no-repeat; }
.block-background-repeat-x{ background: url('../img/avatar.jpg') repeat-x; }
.block-background-repeat-y{ background: url('../img/avatar.jpg') repeat-y; }
.block-background-repeat-space{ background: url('../img/avatar.jpg') space; }
.block-background-repeat-round{ background: url('../img/avatar.jpg') round; }

Приклад 4 - background або background-position - задає початкову положення фонового зображення

Теорія:
- background-position: 0 0; - відступ по горизонталі = 0 і відступ по вертикалі =0
- background-position: 50%; - по центру
- background-position: 0 50%; - зліва
- background-position: 100% 100%; - в нижньому правому вуглі
- background-position: 30px 20px; - відступ по горизонталі = 30px і відступ по вертикалі =20px
Гарачий набір:
bc + Tab = background-color: #fff;
css:
.block-background-position-default{ background: url('../img/avatar.jpg') 0 0 no-repeat; }
.block-background-position-center{ background: url('../img/avatar.jpg') 50% no-repeat; }
.block-background-position-left{ background: url('../img/avatar.jpg') 0 50% no-repeat; }
.block-background-position-corner{ background: url('../img/avatar.jpg') 100% 100% no-repeat; }
.block-background-position-place{ background: url('../img/avatar.jpg') 25px 30px no-repeat; }

block-background-position-center
