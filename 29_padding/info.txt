29 Теорія CSS - відступи padding та margin: 
	- padding - внутрішній відступ від межі об'єкта до його вмісту
		- внутрішні відступи:
			- padding-top; - зверху
			- padding-bottom; - знизу
			- padding-left; - зліва
			- padding-right; - зправа
	- margin - зовнішній відступ від межі об'єкта до межі інших об'єктів
		- зовнішні відступи:
			- margin-top; - зверху
			- margin-bottom; - знизу
			- margin-left; - зліва
			- margin-right; - зправа
	- margin, padding - формат скороченого запису:
		- X - відступ по усьому периметру
		- X Y - відступ: зверху та знизу -X, зліва та зправа - Y
		- X1 Y X2 - відступ: зверху-X1, знизу -X2, зліва та зправа - Y
		- X1 Y1 X2 Y2 - відступи: зверху - X1 та знизу - X2, зправа - Y1 зліва - Y2
	- margin, padding - працюють лише з блочними елементами
 	- box-sizing - алгоритм визначення розміру блоку
		- content-box - (за замовчуванням) розмір блоку = width/height + padding (width, height - розмір контенту)
		- border-box - розмір блоку = width/height (padding входить у розмір блоку - контент стискається)
	- відступи у заливці об'єкта застосовується: 
		- padding: Хpx - заливка фоновим зображенням 
		- padding: Хpx, margin: Хpx; - заливка кольором або градієнтом (застосування margin: Хpx; залежить від дизайну)
	- відступи у позиціонуванні об'єкта застосовується:		 
		- padding: Хpx - при абсолютному позиціонуванні 
		- margin: Хpx; - залежить від дизайну	
	- відкрити приховану частину об'єкту:
		- padding: Хpx - виводить з-за рамки контейнеру
		- margin: Хpx; - залежить від дизайну	
	- центровка обєкта заданої ширини у блоці:
		- margin: 0 auto; - центрує об'єкт у блоці
	- margin - може бути відємним:
		- margin: -50px 0 0 0 - пінявся об'єкт і разом з ним ті, що були за/під ним
		- top: -50px -  - пінявся лише сам об'єкт
		- transform: translate(0, -50px); - пінявся лише сам об'єкт
	- padding - НЕ може бути відємним
	- FLEX-конструкція:
		- колонки у вигляді карток:
			- flex-контейнер:
				- margin: 0 -15px; - розсуває контейнер за межі контейнера контенту
			- flex-колонки
				- ділять flex-контейнер на рівні частини
				- займають весь простір flex-контейнера (включає частину що вийшла за межі контейнара контенту)
				- padding: 0 15px; - ввідсуває з лівого/ правого краю внутрішні межі вмісту колонки
			- flex-елементи:
				- padding: 15px; - робить внутрішній відступ контенту елемента
		- звичайні колонки:
			- flex-контейнер:
				- margin: 0 -15px; - розсуває контейнер за межі контейнера контенту
			- flex-колонки
				- ділять flex-контейнер на рівні частини
				- займають весь простір flex-контейнера (включає частину що вийшла за межі контейнара контенту)
			-	 padding: 0 15px; - ввідсуває з лівого/ правого краю внутрішні межі вмісту колонки
	- FLEX-контейнер: - відкрити приховану частину контенту без зміни його положення:
		- padding: 25px 0; - відкриває приховану частину контенту (при цьому контейнер збільшується і контент змінює своє положення)
		- margin: -25px 0; - піднімає контейнер угору при цьому контент повертається на початкове місце
	- margin - відступ між двома елементами схлопується:
		- (margin-bottom:20px) max (margin-top:60px) = 60px - обидва позитивні, тоді берем максимальний відступ
		- (margin-bottom:-20px) + (margin-top:60px) = 40px - один негативний, тоді складаємо два відступи
		- (margin-bottom:-20px) + (margin-top:-60px) = -60px - обидва негативні, тоді берем відступ максимальний по модулю 
		- батька(margin-top:20px) + нащадка(margin-top:60px) = 60px - відступ нащадка більший за відступ батька:
			- відступ батька = відступу нащадка
			- відступ між батьком та нащадком буде відсутній
	- принципи застосування відступів:
		- відступи потрібно задавати від попереднього елементу до іншого:
			- зверху до низу
			- зліва на право
			- у останнього елемента групи, онулити відступ
		- відступи задаються самому верхньому батьку у конструкції
			- "будинок" задає внутрішній відступ - house > padding: 50px 50px;
			- "тіло списку" задає зовнішній відступ - list__body > margin: 0 0 20px 0;
			- "пункти списку" - задають відступи внутрі пункту - list li > padding: 5px;
		- зовнішні відступи не задають незалежним елементам
			- .header - незалежний блок має лише відступи, що відповідає його внутріньому контенту
			- .contact__header - незалежний блок вклячається як елемент іншого блоку, що включає зовнішній відступ 
		- відступи повинні покращувати зручність користування та сприйняття
			- зайві відступи
			- площа натискання на кнопку
