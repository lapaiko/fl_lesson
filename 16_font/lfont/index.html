Приклад 1 - Підключення шрифтів

Загальні відомості:
head - підключення через link scss - підключення через @import url
- зайти за посиланням https://fonts.google.com
- натиснути кнопку "Фільтр" і вибрати потрібну розкладку (кирилиця/латиниця)
- ввести у пошуку назву потрібного шрифту
- з знайдених вибрати потрібний шрифт
- вибрати потрібні стилі шрифту
- з поля "link" скопіювати посилання і вставити у html перд тегом link
- з поля "import" скопіювати стрічку імпорту і вставити у початок файлу scss
html head:

link rel="preconnect" href="https://fonts.googleapis.com"
link rel="preconnect" href="https://fonts.gstatic.com" crossorigin
link href="https://fonts.googleapis.com/css2?family=PT+Sans:ital,wght@0,400;0,700;1,400&display=swap" rel="stylesheet"


scss:

@import url('https://fonts.googleapis.com/css2?family=PT+Sans:ital,wght@0,400;0,700;1,400&display=swap');

.textoriginal { font-size: 30px;line-height: 35px;
p { margin: 0 0 30px 0;}
}
.text {font-size: 30px;line-height: 35px;
font-family: "PT Sans";
font-style: italic;
font-weight: 700;
p { margin: 0 0 30px 0;}
}

Приклад 2 - Конвертація шрифтів

Загальні відомості:
конвертація шрифта з nomail.com.ua на www.font2web.comшриф беремо з ресурсу nomail.com.ua і конвертуємо на ресурсі www.font2web.com
- зайти за посиланням https://nomail.com.ua/
- змінити посилання на https://nomail.com.ua/font/Назва шрифта
- вибрати з списку знайденого потрібний шрифт
- натиснути кнопку "Завантажити"
- скопіювати файл шрифта у теку "font" свого проекту
- перейти за посиланням http://www.font2web.com/
- натиснути кнопку "Вибрати файл" і вибрати .ttf файл шрифта
- наснути кнопку "Конвертувати та завантажити"
- отриманий архів розпакувати у теку "font" свого проекту
- у теці архіву шрифта відкрити файл "font.scc" скопіювати і вставити на початок scss файлу свого проекту
- поле font-family відкоригувати назву шрифта
- поле font-weight відкоригувати значення відповідно назви шрифта
- поле font-style відкоригувати стиль відповідно назви шрифта
- поле src відкорегувати шляхи до файлів шрифтів
